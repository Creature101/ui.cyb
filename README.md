### How to work with Storybook

Requirements:

yarn version 1.12.3

How to run:

```
yarn install
yarn start
```

For link in project:

```
yarn link
```

in your project

yarn link '@cybercongress/ui'


To publish:

change version in package.json

```
npm login
npm publish
```
